# 💎 คัมภีร์การติดตั้ง OrgChat AI: ฉบับละเอียดที่สุดในโลก (สำหรับมือใหม่)

คู่มือนี้จะพาคุณติดตั้งระบบ "น้องพั้น" ตั้งแต่เครื่องเปล่าๆ จนถึงขั้นใช้งานได้จริง โดยจะอธิบายทุกปุ่มที่ต้องกด ทุกค่าที่ต้องกรอก แบบละเอียดที่สุดค่ะ

---

## 🟢 ตอนที่ 1: เตรียมบ้านให้ AI (การติดตั้งโปรแกรม)

### 1. ติดตั้ง Python (หัวใจของระบบ)
1. เข้าเว็บ [python.org](https://www.python.org/downloads/)
2. กดปุ่มสีเหลือง **Download Python 3.xx.x**
3. **[สำคัญมาก!]** เมื่อเปิดตัวติดตั้งขึ้นมา ให้มองหาช่องที่เขียนว่า **"Add Python to PATH"** (อยู่ด้านล่างสุด) ให้ **ติ๊กถูก** ก่อนกด Install Now
4. รอจนเสร็จ แล้วกด Close

### 2. ติดตั้ง Tesseract OCR (ตาของ AI สำหรับอ่านสลิป)
1. เข้าเว็บ [UB-Mannheim Tesseract](https://github.com/UB-Mannheim/tesseract/wiki)
2. มองหาลิงก์ `tesseract-ocr-w64-setup-5.x.x.exe` แล้วกดดาวน์โหลด
3. ตอนติดตั้ง ให้กด Next ไปเรื่อยๆ จนถึงหน้า **"Choose Components"**
4. มองหาหัวข้อ **"Additional script data"** และ **"Additional language data"** กดเครื่องหมาย + แล้วหาภาษา **"Thai"** ให้ติ๊กถูกทั้งสองที่ (เพื่อให้บอทอ่านภาษาไทยได้แม่นๆ ค่ะ)
5. ติดตั้งจนเสร็จ

## 🟡 ตอนที่ 2: ขออนุญาต Google (เชื่อมต่อ Drive และ Sheets)

ขั้นตอนนี้คือการทำให้บอทสามารถเขียนไฟล์ลง Google Drive ของเราได้ค่ะ

1. เข้าไปที่ [Google Cloud Console](https://console.cloud.google.com/)
2. กด **Create Project** (อยู่มุมบนซ้าย) ตั้งชื่อโครงการอะไรก็ได้ เช่น `My-OrgChat-AI`
3. ในช่องค้นหาด้านบน พิมพ์ว่า **"Google Drive API"** แล้วกดเข้าไป กดปุ่ม **Enable**
4. ทำเหมือนเดิม ค้นหา **"Google Sheets API"** แล้วกด **Enable**
5. ไปที่เมนู **APIs & Services > Credentials**
6. กดปุ่ม **+ Create Credentials** เลือก **Service Account**
7. ตั้งชื่อบอท (เช่น `punch-bot`) แล้วกด **Create and Continue** จนเสร็จ
8. คุณจะได้ **อีเมลยาวๆ** มาตัวหนึ่ง (เช่น `punch-bot@project-id.iam.gserviceaccount.com`) **[จดไว้!]**
9. กดเข้าไปที่อีเมลนั้น ไปที่แท็บ **Keys** > **Add Key** > **Create New Key** > เลือก **JSON**
10. จะมีไฟล์โหลดลงเครื่องเรา ให้เปลี่ยนชื่อไฟล์นั้นเป็น **`service-account.json`** แล้วเอาไปวางในโฟลเดอร์ `webai` ของโปรแกรมเราค่ะ

**[ขั้นตอนนี้ห้ามลืม!]**
- ไปที่ Google Drive ของคุณ สร้างโฟลเดอร์ใหม่ที่จะให้บอทเก็บรูป
- กด **Share** โฟลเดอร์นั้น แล้วเอา **อีเมลยาวๆ** จากข้อ 8 มาใส่ ให้สิทธิ์เป็น **Editor**
- ทำแบบเดียวกันกับไฟล์ Google Sheet ที่จะใช้เก็บข้อมูลด้วยนะคะ

## 🟠 ตอนที่ 3: เปิดสมอง AI (สมัคร Gemini API)

1. เข้าไปที่ [Google AI Studio (Gemini)](https://aistudio.google.com/)
2. ล็อกอินด้วย Gmail ของคุณ
3. กดปุ่ม **Get API Key** ทางซ้ายมือ
4. กด **Create API Key in new project**
5. ก๊อปปี้รหัสที่ได้ (รหัสยาวๆ) มาเก็บไว้

---

## 🔵 ตอนที่ 4: เชื่อมต่อ LINE (ขอรหัส Webhook)

1. เข้าไปที่ [LINE Developers Console](https://developers.line.biz/)
2. สร้าง **Provider** (ชื่อบริษัทเรา) และสร้าง **Messaging API Channel** (ใส่รูปบอท ชื่อบอท)
3. ไปที่แท็บ **Messaging API**:
   - เลื่อนลงมาล่างสุดที่ **Channel access token** กด **Issue** เพื่อรับรหัสยาวๆ **[ก๊อปปี้ไว้!]**
4. ไปที่แท็บ **Basic settings**:
   - มองหา **Channel secret** **[ก๊อปปี้ไว้!]**

---

## 🔴 ตอนที่ 5: การตั้งค่า "กุญแจ" ในโปรแกรม (.env)

ให้เปิดไฟล์ที่ชื่อ **`.env`** ในโฟลเดอร์โปรแกรมด้วย Notepad แล้วเอาค่าที่จดไว้มาใส่หลังเครื่องหมาย `=` ดังนี้:

- `GEMINI_API_KEY=` (ใส่รหัสจากตอนที่ 3)
- `LINE_CHANNEL_ACCESS_TOKEN=` (ใส่รหัสจากตอนที่ 4 ข้อ 3)
- `LINE_CHANNEL_SECRET=` (ใส่รหัสจากตอนที่ 4 ข้อ 4)
- `SPREADSHEET_ID=` (รหัสที่อยู่บนลิงก์ Google Sheet ของคุณ)
- `PARENT_FOLDER_ID=` (รหัสที่อยู่บนลิงก์โฟลเดอร์ Google Drive ของคุณ)

---

## 🚀 ตอนที่ 6: การเดินเครื่องน้องพั้น

1. เปิดหน้าจอดำๆ (Command Prompt) ในโฟลเดอร์โปรแกรม
2. พิมพ์คำสั่งนี้เพื่อลงทะเบียนโปรแกรม (ทำครั้งเดียว):
   ```bash
   pip install -r requirements.txt
   ```
3. สั่งรันน้องพั้น:
   ```bash
   python app_server.py
   ```
4. หากต้องการให้ LINE คุยกับเครื่องเราได้ (กรณีรันที่บ้าน) ให้ใช้โปรแกรม **Ngrok** เพื่อทำลิงก์สาธารณะ แล้วเอาลิงก์นั้นไปใส่ในช่อง **Webhook URL** ในหน้า LINE Developers ค่ะ
